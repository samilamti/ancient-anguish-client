# v4.0 Release Notes

**Date:** March 14, 2026
**Previous release:** v3.3 (March 12, 2026)

---

## Highlights

This release significantly expands the client's immersion and usability features: a unified area configuration system, emoji rendering, clickable hyperlinks, TAB word completion, prefix-filtered command history, audio crossfade with playlists, and major improvements to the social windows and area naming systems.

---

## New Features

### Unified Area Configuration
- All area settings (coordinates, background images, music) merged into a single file with heading-per-area format
- New unified Area Configuration screen replaces the separate Audio Settings and Background Image Settings screens
- Automatic migration from old separate config files on first load

### Emoji Rendering
- Text emoticons are now automatically converted to Unicode emoji in terminal output

### Clickable Hyperlinks
- URLs in terminal output are detected and rendered as clickable links
- Links open in the system browser; display text strips the protocol prefix

### TAB Completion
- Pressing TAB in the input bar completes words using capitalized words seen in MUD output
- Repeated TAB presses cycle through matches; any other key resets completion

### Prefix-Filtered Command History
- When text is already typed in the input bar, pressing Up/Down only cycles through history entries that start with that prefix
- Empty input cycles all history as before
- Any non-arrow key or Enter resets the search session

### Unmapped Area Detection & Naming
- Detects when the player is at unmapped coordinates after 3 directional movement commands
- Pulsing gold "Name this area" button appears in the status bar
- Opens a modal dialog for entering the area name
- Blocked during active combat with a snackbar message
- Non-movement commands (look, say, etc.) no longer inflate the counter

### Social Window Improvements
- **Unread indicators**: amber dot and amber text on inactive tabs; amber title bar tint on separate panels
- **Link integration**: clickable hyperlinks rendered within social messages
- **Battle suppression**: temporarily ignores battle patterns during known non-combat HP-loss events (e.g. headache)
- Social message history now persists across sessions

### Audio Enhancements
- **Crossfade**: smooth audio transitions between areas
- **Playlists**: support for multiple music tracks per area with sequential cycling
- **Town music delay**: configurable delay before town music starts playing

### Terminal Readability
- Terminal text rendered with a subtle shadow for crispness against any background
- Blue brightened for better readability on dark backgrounds

### D-Pad Enhancement
- Compass rose center now displays the current area name when known

---

## UI/UX Changes

### Immersions (Trigger Rebrand)
- "Triggers" renamed to "Immersions" across all UI text (settings, title bars, help dialogs, tooltips)
- Config file on disk renamed to section-based format with Highlights, Sounds, and Gags sections
- Aliases config also reformatted to Markdown

### Area Configuration Screen
- Single unified screen for managing area coordinates, background images, and music
- Replaces separate Audio Settings and Background Image Settings screens

---

## Bug Fixes

- **Audio crash**: Fixed crash from premature source disposal during crossfade operations
- **Emoticon conflict**: Removed `(n)` emoticon mapping that interfered with sending "north" command
- **Area detection priority**: Text-based area detection now defers to coordinate-based lookup when coordinates are available
- **Audio reactivity**: Fixed area audio not updating immediately when area assignment changes
