# v4.1 Release Notes

**Date:** March 15, 2026
**Previous release:** v4.0 (March 14, 2026)

---

## Highlights

Bug fixes for emoji parsing and social history, plus a new cross-platform window attention feature that flashes the taskbar/dock when receiving tells while alt-tabbed.

---

## New Features

### Window Attention on Incoming Tells
- Taskbar/dock flashes when an incoming Tell arrives while the window is not focused
- Windows: taskbar icon flashes until the window is clicked (FlashWindowEx)
- macOS: dock icon bounces once (NSApplication.requestUserAttention)
- Linux: taskbar urgency hint (gtk_window_set_urgency_hint)
- Custom platform channel — no new package dependencies

---

## Bug Fixes

- **Emoji in URLs**: Emoticon patterns inside URLs (e.g. `://`) are no longer replaced with emoji
- **Social history ordering**: Chat and tell history writes are now serialized via queues to prevent continuation lines from being written out of order
- **Social tab scroll**: Message list now scrolls to bottom on initial build after a tab switch
