# v5.1 Release Notes

**Date:** March 16, 2026
**Previous release:** v5.0 (March 16, 2026)

---

## Highlights

Social window polish and scrollback simplification. Chat and tell panels now show cleaner, reformatted messages, and the unused scrollback buffer setting has been removed in favor of a fixed 5,000-line limit.

---

## Improvements

### Social Windows
- Chat messages strip the `[Chat] ` prefix for cleaner display
- Incoming tells reformatted from "Foo tells you: hi" to "Foo: hi"
- Outgoing tells reformatted from "You tell Foo: hi" to "➡️ Foo: hi"
- Social messages now pass through to the terminal when panels are hidden (gag only applies when visible)
- Default panel width increased from 340 to 480 pixels

### StyledLine Utilities
- New `subLine()`, `removeRange()`, and `prepend()` methods for character-level manipulation of styled terminal lines

---

## Removed

### Scrollback Buffer Setting
- Removed the scrollback buffer slider from Settings (was 1k–100k range)
- Removed `scrollbackLines` from `AppSettings` model, serialization, and constants
- Buffer is now a fixed 5,000 lines (previously hardcoded at 10,000 regardless of setting)
