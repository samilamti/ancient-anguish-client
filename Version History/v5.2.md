# v5.2 Release Notes

**Date:** March 17, 2026
**Previous release:** v5.1 (March 16, 2026)

---

## Highlights

Social history reorganized to per-day files, command history capped, and TAB completion expanded to include all words. Five pre-existing test failures fixed, bringing the suite to 669/669 passing.

---

## Improvements

### Social History Storage
- Chat and tell history now stored in per-day files under `Chat History/` and `Tell History/` subfolders (e.g. `Chat History/2026-03-17.md`)
- On startup, today's previous messages are loaded into the social windows
- Old single-file history files (`Chat History.md`, `Tell History.md`) left in place as archives

### Command History
- `Command History.md` now capped at 20 entries on disk (previously unlimited growth)

### TAB Completion
- Now captures all 3+ letter words from MUD output (previously only capitalized words)
- List remains sorted by most recently seen

---

## Bug Fixes

### Test Suite
- Fixed widget test: override audio provider with stub to avoid SoLoud DLL load in test runner
- Fixed trigger rules tests: corrected rule IDs from stale `trig_tells` to actual `hl_01`
- Fixed game state test: await FutureProvider resolution before asserting area lookup
- All 669 client tests and 62 server tests now pass

---

## Technical Details

- Server file whitelist updated with regex patterns for new `Chat History/` and `Tell History/` subfolder paths
- `CommandHistoryService.maxEntries` constant used by both disk persistence and in-memory cap
- 669 client tests passing, 62 server tests passing, 0 analysis errors
