# v5.3 Release Notes

**Date:** March 19, 2026
**Previous release:** v5.2 (March 17, 2026)

---

## Highlights

Social windows polished with a new autocomplete recipient field and lazy-loading message display with alternating-row striping.

---

## Improvements

### Tell Recipient Autocomplete
- Replaced the dropdown/chip selector with an inline autocomplete text field showing full character names
- Type to filter recent tell partners; select from the dropdown or type a new name
- Dropdown arrow integrated as a suffix icon inside the text field

### Social Message List Performance
- Messages now lazy-load: only the 50 most recent are shown initially
- Scrolling to the top loads 50 more at a time with scroll position preserved
- A "scroll up for older messages" indicator shows how many remain
- Auto-scroll behavior unchanged — new messages still snap to the bottom

### Pajama Striping
- Even-indexed message rows display a subtle tinted background for visual separation
- Improves scanability in busy chat and tell panels

### README
- Added social windows, text selection, clickable links, and web support to the feature list
- Fixed scrollback description to reflect actual 5,000-line buffer
- Added Web to supported platforms

---

## Technical Details

- `social_input_bar.dart`: `RawAutocomplete<String>` with `recentTellPartnersProvider` for name suggestions
- `social_message_list.dart`: `_displayCount` / `_batchSize` pagination, `primary.withAlpha(10)` even-row tint
- 669 client tests passing, 62 server tests passing, 0 analysis errors
