# v6.0 Release Notes

**Date:** March 20, 2026
**Previous release:** v5.3 (March 19, 2026)

---

## Highlights

Configurable font size across the entire UI, multi-line input with Shift+Enter, and macOS file access fixes.

---

## Improvements

### Configurable Font Size
- New slider in Settings (8–32pt, default 14pt) controls all terminal, social panel, and input bar text
- Terminal font metrics automatically remeasured when size changes for accurate text selection
- Social message timestamps and hints scale proportionally

### Multi-line Input
- Enter sends the command; Shift+Enter inserts a newline
- Terminal input expands up to 5 lines; social input up to 3 lines
- Works in both the main terminal input bar and social panel input bars

### macOS Sandbox Entitlements
- Added user-selected and downloads read-write file access permissions
- Fixes background image and audio file loading on macOS

### Analysis Configuration
- Flutter analyzer now excludes the `server/` directory, which has its own standalone Dart analysis

### Release Workflow Skill
- New `/release` composite skill chains memory updates, logical commits, release notes, push, and tagging into one workflow

---

## Technical Details

- `settings_provider.dart`: `fontSize` field (default 14.0, clamped 8–32) with `setFontSize()` method
- `terminal_view.dart`: `_measuredFontSize` tracking for font metric invalidation
- `input_bar.dart` / `social_input_bar.dart`: `KeyEvent` handler for Enter/Shift+Enter, `maxLines` support
- 669 client tests passing, 62 server tests passing, 0 analysis errors
