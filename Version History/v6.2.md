# v6.2 Release Notes

**Date:** March 21, 2026
**Previous release:** v6.1 (March 20, 2026)

---

## Highlights

Party emotes, social window filtering, area renaming, and new message highlight animation.

---

## New Features

### Party Emote Support
- Party messages now distinguish between say and emote formats
- Say: `<Party> Character : message` → `Character: message`
- Emote: `<Party> Character does something` → `Character does something`
- Party kill system messages (`Your party just killed...`) are filtered out

### Area Renaming
- Click the area name in the status bar to rename it via a dialog
- D-pad center area name is also tappable to rename
- Rename transfers all coordinates, backgrounds, and music to the new name

### New Message Highlight
- Incoming social messages briefly flash with a highlight animation for visibility

---

## Improvements

### Social Window Filtering
- NPC announcements (`The High Priest: ...`) no longer appear in social windows
- Status broadcasts (`begins exploring.`, `leaves to explore elsewhere.`) are filtered out
- Multiple spaces in social messages are collapsed to single spaces for cleaner display

### Auto-Activate Social Tab
- In tabbed mode, the active tab auto-switches to match incoming message type when the input bar has focus

### Tell Recipient Memory
- Last tell recipient is restored from history on startup, so reply works immediately

### Chat History Format
- Chat history files no longer store the redundant `[Chat]` prefix

---

## Technical Details

- `social_message_parser.dart`: `_partySayRegex` / `_partyEmoteRegex` replace single `_partyRegex`; `isPartySystemMessage()` and `isChatSystemMessage()` filter methods
- `StyledLine.collapseSpaces()` in `styled_span.dart` for multi-space normalization
- `showRenameAreaDialog()` extracted to shared function in `status_bar.dart`, used by both status bar and D-pad
- `UnifiedAreaConfigManager.renameArea()` transfers all config entries atomically
- `_MessageWidget` converted to `StatefulWidget` with `AnimationController` for highlight effect
- `TerminalBlock` keyed by `startLineIndex` for stable `ListView` identity
- 681 client tests passing, 62 server tests passing, 0 analysis errors
