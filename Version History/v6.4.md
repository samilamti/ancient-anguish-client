# v6.4 Release Notes

**Date:** April 6, 2026
**Previous release:** v6.3 (March 21, 2026)

---

## Highlights

Cross-platform build support with macOS verified, social panel UX improvements, and codebase-wide line ending normalization.

---

## Improvements

### Social Panel Enhancements
- **Keyboard shortcuts**: Ctrl/Cmd+1/2/3 switches between social tabs from both the input bar and terminal view
- **Pulsing unread indicator**: Unread tabs now show an animated pulsing amber dot (opacity 0.4-1.0, 1200ms cycle) instead of a static indicator
- **Active tab styling**: Subtle primary background tint, thicker bottom border, and larger font for the active tab

### NPC Tell Filtering
- Tells from NPCs (e.g., Priest) now stay in the main terminal instead of being routed to the Tells social tab
- Added `isTellNpc()` with a configurable NPC sender set in `social_message_parser.dart`

### Auto-Tab Throttle
- Added 2-second cooldown on social tab auto-switching to prevent rapid tab flipping during message bursts

### Cross-Platform Build Scripts
- `build.sh`, `run.sh`, and `build-release.sh` now auto-detect the host OS via `uname -s` and build for macOS, Linux, or Windows accordingly
- `build-release.sh` uses platform-appropriate packaging (zip for macOS .app, tar.gz for Linux, PowerShell for Windows)
- `.claude/launch.json` updated to invoke `flutter` directly instead of through `cmd.exe`

### CocoaPods Integration
- Added Podfile for both iOS and macOS to support native plugin dependencies (flutter_soloud, file_picker, url_launcher)

### Line Ending Normalization
- Normalized all 268 tracked files from CRLF to LF for cross-platform consistency
